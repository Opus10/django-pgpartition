django-pgpartition
########################################################################

Documentation
=============

`View the django-pgpartition docs here
<https://django-pgpartition.readthedocs.io/>`_.

Installation
============

Install django-pgpartition with::

    pip3 install django-pgpartition

After this, add ``pgpartition`` to the ``INSTALLED_APPS``
setting of your Django project.
